* 9대 사서


### 로어

```
'생육하여라'

그럴듯한 피카소의 복사본... 피카소의 원본보다 20년 전에 그려졌다. 브라이언 레빈센이 다시 한 번 장난을 쳤다.
```


```
'빨강, 파랑'

그럴듯한 피카소의 복사본... 피카소의 원본보다 20년 전에 그려졌다. 브라이언 레빈센이 다시 한 번 장난을 쳤다.
```

```
'또 다시, 밤중에'

레빈센 박사의 '예견들' 중 하나. 키르히너가 20년 후에 그린 그림의 정확하고 (불가능한) 복사본으로, 레빈센은 종종 그의 거울 꿈을 통해 키르히너의 작업을 예측했다.
```


```
'목소리를 듣는 자들에게'

레빈센의 '예견들', <i>실제</i> 예술가가 20년은 있어야 캔버스에 그릴 그림들의 복사본이다.
```

```
'세도나, 아직 그려지지 않은'

레빈센 박사의 '예견들' 중 하나로, 그려지지 않은 그림의 불가능한 복사본이다. 그는 그것에 광사학의 인장과 함께 '1958년'이라는 년도를 남겼다.
```

```
'조용한 곳에서의 사랑'

레빈센 박사의 '예견들' 중 하나. 키르히너가 20년 후에 그린 그림의 정확하고 (불가능한) 복사본으로, 레빈센은 종종 그의 거울 꿈을 통해 키르히너의 작업을 예측했다.
```
