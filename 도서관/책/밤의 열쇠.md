```
밤의 열쇠

한낮의 셰르보 호텔 지하실에서 이루어진 이 논의는 1900년 전후로 빈 주점에서 기록된 '저울의 벌레'라는 성명문으로 이어졌다.
```
### 1장

```
내가 읽고 있는 내용은...

'시민의 칼'이나 '한밤의 순례자'처럼 거창한 별명을 써대는 벌레들은 그들이 이전에 전개한 모든 운동이 무모하고 위험했다는 것을 인정하며, 인간이 있기 이전의 시간들, 즉 돌에서 온 시간들을 복귀시키는 것만이 '영원의 폭정'을 끝내는 진정한 방법임에 찬성한다.
```
### 2장

```
내가 읽은 내용은...

벌레들은 수레파퀴와 파도, 부싯돌, 술로챠나, 거미의 문, 그나토베이시스 각하, '꺼지지 않는 불' 위켈을 '행위의 선전을 통해 모든 역사로 돌려놓아야 할' 일곱 개의 필수적인 펜티멘토라 규정했다. 목록에는 다소 이상한 이름이 끼어 있으며, 명백히 무언가의 타협을 통해 나온 결과물로 보인다. 이에 대해 논의하는 과정은 모든 참가자들을 녹초로 만들어버린 한편, 참가자들은 '틈새구멍'이라는 모호한 언급 외에는 필요한 행위에 대해 구체적인 설명을 제시하지 않고 있다.
```
