```
연기 속의 형상

앰브로즈 웨스트콧이 연초 블렌딩하는 법과 '파이프의 신탁'에 대해 설명한 내용들을 담고 있다.
```
### 1장

```
내가 읽고 있는 내용은...

책의 곳곳에 불티로 인해 생긴 작은 구멍들이 흩뿌려져 있다...
```
### 2장

```
내가 읽은 내용은...

웨스트콧은 마치 격언처럼, '거짓이 불태워지면, 지혜가 솟아오른다. <i>반대의 경우도 마찬가지다.</i>' 라고 말하며 파이프가 파수자의 나무에 핀 붉은 꽃을 상징한다고 말한다.
```
